How to airgait¶

This tutorial will help you to airgait with the March exoskeleton.

Airgait on the exoskeleton directly¶

The setup¶

  • Screen connected with an HDMI cable to the March exoskeleton.
  • The March exoskeleton wireless keyboard and mouse.
  • Exoskeleton hanging in the cage.
  • Tools needed to power the March exoskeleton.

Start up¶

  1. High voltage: Turn the high voltage on by releasing the emergency button and enabling power.
  2. Start the exoskeleton pc: Keep the start button pressed until the blue light is continuously on. The computer is now booting. The display, mouse and keyboard should automatically connect.
  3. Start the software: Type in a sourced terminal roslaunch march_launch march.launch.
  4. Select launch settings: You can select launch settings by appending launch options to the launch. use roslaunch --ros-args march_launch march.launch to see what options are available. See the march_launch page for more info on how to launch the exoskeleton.

Remote airgait¶

Note

For this part you have to complete the Preparation before connecting.

The setup¶

  • Your pc.
  • Exoskeleton hanging in the cage.
  • Tools needed to power the March exoskeleton.
  • WiFi Router

Start up¶

  1. High voltage: Turn the high voltage on by releasing the emergency button and enabling power.
  2. Start the exoskeleton pc: Keep the start button pressed until the blue light is continuously on. The computer is now booting. The computer will automatically connect with the WiFi router.
  3. Start your computer Start your computer and connect with the WiFi router.
  4. SSH: Start a terminal to connect with the exoskeleton pc. Connect with ssh march@march.
  5. Start the software: Type roslaunch march_launch march_headless.launch.
  6. Input Device: Type in a sourced terminal on your computer roslaunch march_rqt_input_device march_rqt_input_device.launch to start the input device.

Note

For this part we assume that on the pc you are running everything is installed described in Create your workspace
